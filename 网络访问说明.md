## 🧪 从本地电脑调用 API

### cURL 示例

```bash
# 替换 YOUR_SERVER_IP 为服务器实际 IP
SERVER_IP="YOUR_SERVER_IP"

# OpenAI 兼容接口
curl -X POST "http://$SERVER_IP:8765/v1/chat/completions" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-3.5-turbo",
    "messages": [{"role": "user", "content": "Hello!"}]
  }'

# Anthropic 兼容接口
curl -X POST "http://$SERVER_IP:8765/v1/messages" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "claude-3-haiku-20240307",
    "messages": [{"role": "user", "content": "Hello!"}],
    "max_tokens": 1024
  }'
```

### Python 示例

```python
import httpx

# 服务器地址
SERVER_IP = "YOUR_SERVER_IP"
BASE_URL = f"http://{SERVER_IP}:8765"

# OpenAI 兼容接口
response = httpx.post(
    f"{BASE_URL}/v1/chat/completions",
    json={
        "model": "gpt-3.5-turbo",
        "messages": [{"role": "user", "content": "Hello!"}]
    }
)
print(response.json())
```

### OpenAI SDK 示例

```python
from openai import OpenAI

# 指向你的服务器
client = OpenAI(
    api_key="not-needed",  #  proxy 不强制验证
    base_url="http://YOUR_SERVER_IP:8765/v1"
)

response = client.chat.completions.create(
    model="gpt-3.5-turbo",
    messages=[{"role": "user", "content": "Hello!"}]
)
print(response.choices[0].message.content)
```

## 🔐 安全注意事项

⚠️ **重要：** 现在服务对公网开放，请注意安全：

1. **不要泄露 API Key** - `.env` 文件已加入 `.gitignore`，不要上传到公开仓库

2. **考虑添加访问限制** - 如果可能，配置防火墙只允许特定 IP 访问：
   ```bash
   # 示例：只允许特定 IP 访问 8765 端口
   sudo firewall-cmd --add-rich-rule='rule family="ipv4" source address="YOUR_LOCAL_IP/32" port port="8765" protocol="tcp" accept'
   ```

3. **使用 HTTPS** - 生产环境建议使用反向代理 (Nginx) 配置 HTTPS

4. **监控日志** - 定期检查 `proxy.log` 查看异常访问

## 🆘 故障排除

### 无法从本地访问

1. 检查服务器防火墙：
   ```bash
   sudo firewall-cmd --list-all
   ```

2. 检查端口是否监听：
   ```bash
   netstat -tlnp | grep 8765
   ```
   应该看到 `0.0.0.0:8765`

3. 测试服务器本地访问：
   ```bash
   curl http://localhost:8765/health
   ```

### 服务意外停止

查看日志：
```bash
cat /home/XXBAi/HKUST/router/proxy.log
```

重启服务：
```bash
cd /home/XXBAi/HKUST/router
nohup uvicorn app.main:app --host 0.0.0.0 --port 8765 > proxy.log 2>&1 &
```

### API Key 无效

确保 `.env` 文件配置正确：
```bash
cat /home/XXBAi/HKUST/router/.env
```

修改后重启服务：
```bash
pkill -f "uvicorn app.main:app"
cd /home/XXBAi/HKUST/router
nohup uvicorn app.main:app --host 0.0.0.0 --port 8765 > proxy.log 2>&1 &
```
